# Spec Template

## Problem
Describe the business problem and desired outcome.

## Data Needed
List Clockify entities and external data required.

## Triggers
Events or user actions that start flows (webhooks, lifecycle, UI actions).

## UI Surface
Which components/tabs/views are used and what they show.

## External Systems
List external APIs/services, auth pattern, and data mapping.

## Configuration
Settings (structured) and any secrets. Who can edit them.

## Edge Cases
Failures, retries, rate limits, permissions, and recovery steps.
